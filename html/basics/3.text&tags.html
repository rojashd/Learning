Structuring the text with tags


Sometimes we want to format the paragraph with line-breaks.

For example:

"< html >
< body >
< h1 > My favorites things < /h1 >
< p >
Raindrops on roses Whiskers on kittens Bright copper kettles Warm woolen mittens
< /p >
< /body >
< /html >"

This song would look much better if "Raindrops on roses" appeared on one line, Whiskers on kittens on the next, and so on.


"< html>
< body>
< h1> My favorites things < /h1>
< p>
Raindrops on roses < br >
Whiskers on kittens < br >
Bright copper kettles < br >
Warm woolen mittens < br >
< /p>
< /body>
< /html>"

We can separate lines with the help of the line break tag: "< br>"

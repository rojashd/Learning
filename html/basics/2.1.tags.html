Opening tags

An Opening Tags "< p >" begins a section of a page content.


Ending tags

A closing/ending Tag "< /p >" ends a section of a page content. Closing tags always proceed the element with a "/".


For example:

The elemnt "< p > < /p >" is commonly used to markup a section of text, you would open the element with the Opening paragraph tag "< p >" and close it with the closing/endindg paragraph tag "< /p >".


Empty tags

Thsese are tags that do not contain any closing tag. They only contain the opening tag but they perform some action into the webpage.
For example: the "< br >" line break element, which represents a line break in the webpage elements.
